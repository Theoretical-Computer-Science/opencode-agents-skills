## Key Concepts

### ASO Elements
1. **App Name**: 30-char limit, primary keywords
2. **Subtitle**: 30-char, secondary keywords
3. **Keywords**: 100-char field (iOS), no duplicates
4. **Description**: Hook users, include features
5. **Icon**: Memorable, scalable, testing
6. **Screenshots**: Feature-first, show value
7. **Preview Video**: 15-30 seconds, auto-play

### Keyword Research
- Analyze competitors
- Use ASO tools (Sensor Tower, AppTweak)
- Long-tail keywords have less competition
- Consider local languages
- Update regularly based on data

### Conversion Optimization
- A/B test screenshots and icons
- Feature key benefits in first 3 screenshots
- Localize for target markets
- Maintain high ratings through quality
- Respond to reviews professionally

### Ranking Factors
- Download velocity
- User ratings and reviews
- Engagement metrics
- Keyword relevance
- Backlinks (for web presence)
