## What I do
- Evaluate framework options
- Assess library quality
- Consider community support
- Evaluate maintenance status
- Check licensing
- Assess performance
- Consider learning curve
- Plan migration strategies

## When to use me
When choosing frameworks or libraries.

## Evaluation Criteria
```
Technical Factors:
- Performance benchmarks
- Memory usage
- Bundle size
- Browser support
- TypeScript support

Community Factors:
- GitHub stars
- Last commit date
- Issue response time
- Stack Overflow activity

Maintenance Factors:
- Release frequency
- Version stability
- Deprecation policy
- Migration guides

Business Factors:
- License cost
- Support options
- Vendor lock-in
- Long-term viability
```
