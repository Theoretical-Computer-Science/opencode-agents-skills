## Key Concepts

### Periodic Table Structure

- Groups (1-18): Similar valence electron configuration
- Periods (1-7): Same shell number
- Blocks: s, p, d, f based on orbital filling

### Acid-Base Concepts

**Arrhenius**
- Acid: H⁺ donor
- Base: OH⁻ donor

**Bronsted-Lowry**
- Acid: Proton donor
- Base: Proton acceptor

**Lewis**
- Acid: Electron pair acceptor
- Base: Electron pair donor

### Common Inorganic Compounds

- Acids: HCl, H₂SO₄, HNO₃, H₃PO₄
- Bases: NaOH, KOH, Ca(OH)₂
- Salts: NaCl, CaCO₃, NH₄NO₃
- Oxides: CO₂, SiO₂, Al₂O₃
